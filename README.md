# Скидки Золотого Яблока

Веб-приложение для отслеживания товаров со скидками на goldapple.ru.

## Возможности

- Автоматический сбор товаров со скидками из каталога Золотого Яблока
- Веб-интерфейс с фильтрами по категории, бренду, размеру скидки
- Сортировка по скидке, цене, названию
- Автоматическое обновление данных раз в сутки

## Установка

```bash
bun install
npx playwright install chromium
```

## Запуск

```bash
bun start
```

Сервер запустится на http://localhost:3000.
При первом запуске автоматически начнётся сбор данных.

## Ручной запуск парсера

```bash
bun run scrape
```

## Технологии

- **Bun** — рантайм и HTTP-сервер
- **Playwright** — headless browser для обхода защиты от ботов
- **Vanilla HTML/CSS/JS** — фронтенд
